About Crossplane

Crossplane is an open-source Kubernetes add-on that enables you to manage infrastructure using custom resources. It allows you to define and manage cloud resources, such as AWS S3 buckets, Google Cloud Storage buckets, Azure Blob Storage, and more, directly from Kubernetes.

With Crossplane, you can treat infrastructure as code, applying the same GitOps principles to your infrastructure that you do to your applications.
